# Beyond Origins

RPG (role-playing video game) written in Java, which is not using any sort of game engine - <b>everything is from scratch.</b> 
The primary goal right now is to have a viable framework that is highly configurable and easy to modify. This is in the <a href="https://github.com/egartley/beyond-origins/tree/master/src/net/egartley/gamelib">gamelib</a> package.

<img src="https://raw.githubusercontent.com/egartley/media/master/screenshots/beyond-origins.png">
